# 백운마을 귀농체험

화순군 백운마을의 귀농 체험 프로그램을 소개하는 웹사이트입니다.

## 기술 스택

- Vue 3
- Vuetify 3
- Vue Router
- Vite

## 설치 방법

```bash
# 프로젝트 클론
git clone [repository-url]

# 종속성 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build
```

## 주요 기능

- 귀농 준비 가이드
- 지역별 정보 제공
- 성공 사례 소개
- 문의하기
- 관련 기관 링크

## 프로젝트 구조

```
src/
├── components/     # 재사용 가능한 컴포넌트
├── views/          # 페이지 컴포넌트
├── router/         # 라우팅 설정
├── plugins/        # Vuetify 설정
└── App.vue         # 루트 컴포넌트
```
